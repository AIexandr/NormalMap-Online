Height map

This texture contains the distance "height" from "floor".
The brighter the pixel, the further away (more depth).
In this application it doesn't matter if it's inverted, the most important thing is the difference between the pixels.
Ideally, a texture with a power of 2 inside (2,4,8,16 etc.)


Using a height map for example as displacement, you can easily see the effect on the surface.
An example heightmap (left), bumps going in (top-right) and bumps going out (bottom-right).
